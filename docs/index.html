    This reference is prepared for APIs required by Smart Operation Platform.

    Base URL: https://orfeostorysite.com/smat_mobile/api

    Version: 1.0

    Example API: https://orfeostorysite.com/smat_mobile/api/{endpoint}

    The authentication process can be conducted in three methods such as, GET, POST, and Header
    You must choose any of the above based on the requirements. Example for each of these methods are also provided.
    GET:
    url:https://orfeostorysite.com/smat_mobile/api/{endpoint}/?token={your token}
    POST:
    body{
    "token":{your token}
    }
    Header:
    header{
    Authorization: Bearer [your token]
    }

    HTTP Request

    POST post_favorite

    Definition

    This API enables users to add products and categories to their list of favorites.

    Product ID to be submitted upon user's clicking on a product

    Category ID to be submitted upon user's clicking on a Category.

    For Product used Type: 1.

    For Category used Type: 2.

    A single Object ID or multiple IDs can be sent (in an Array or int on post body) .

    Authentication Requirement

    Authentication is required for this API.
